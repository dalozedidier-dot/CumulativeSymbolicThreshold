# requirements/

| File | Purpose |
|------|---------|
| `../requirements.txt` | Core runtime dependencies (used by default) |
| `requirements-qcc.txt` | Extra dependencies for QCC (quantum/contextual computing) variant |
| `requirements-qcc-stateprob.txt` | Extra dependencies for QCC state-probability variant |

For the standard ORI-C pipeline, `pip install -r requirements.txt` (root) is sufficient.
