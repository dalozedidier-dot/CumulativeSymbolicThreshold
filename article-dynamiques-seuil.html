Dynamiques à seuil et bascules de régime

Au-delà d'un seuil critique d'accumulation symbolique, le canal de transmission devient auto-renforçant — une transition de phase mesurable.

Seuils

L'hypothèse centrale

Le cadre ORI-C propose une hypothèse testable et falsifiable : au-delà d'un seuil critique d'accumulation symbolique S*, le canal de transmission symbolique devient auto-renforçant. Ce basculement constitue une transition de phase mesurable, d'un régime pré-seuil vers un régime cumulatif symbolique.

Transition de phase

La transition n'est pas graduelle. Elle marque un changement qualitatif dans la dynamique du système : ce qui était linéaire devient non-linéaire, ce qui était réversible devient irréversible.

Variables fondamentales

Le modèle repose sur un ensemble de variables définies ex ante :

  • O(t) — Organisation : proxy de la coordination structurelle et fonctionnelle
  • R(t) — Résilience : capacité d'absorption des perturbations
  • I(t) — Intégration : cohérence inter-composants
  • Cap(t) — Capacité : O(t) · R(t) · I(t)
  • Σ(t) — Mismatch : max(0, D(E(t)) − Cap(t))
  • S(t) — Stock symbolique : répertoire transmissible
  • C(t) — Variable d'ordre : gain symbolique intergénérationnel

Le critère de détection du seuil

La détection du seuil repose sur un critère statistique pré-enregistré :

ΔC(t) > μ_ΔC + k · σ_ΔC    pour m pas consécutifs
k = 2.5, m = 3 (pré-enregistrés)

Ce critère identifie le moment où la variation de C dépasse significativement sa moyenne historique, de façon persistante. Le seuil n'est pas arbitraire : il émerge des données selon une procédure reproductible.

Les trois régimes dynamiques

Régime C(t) S(t) Interprétation
Pré-seuil ≈ 0 Non cumulatif La transmission symbolique ne s'auto-amplifie pas
Transition Instable Décorrélation partielle O, R, I et C commencent à se découpler
Cumulatif > 0, stable Amplificateur Le gain intergénérationnel est verrouillé

La chaîne causale

Le modèle spécifie une chaîne causale explicite :

O(t) ─┐
R(t) ─┼──► Cap(t) = O·R·I ──► Σ(t) = max(0, D − Cap) ──► V(t)
I(t) ─┘

S(t) ──────────────────────────────────────────────────► C(t) ──► ΔC(t) ──► seuil?

Cette architecture permet de tester indépendamment chaque maillon de la chaîne. Si un maillon ne se comporte pas comme prédit, l'hypothèse est falsifiée à cet endroit précis.

Implications pour les systèmes réels

La dynamique à seuil n'est pas une métaphore. Elle décrit un mécanisme observable dans de nombreux systèmes :

  • En économie : les crises ne sont pas des événements linéaires, mais des bascules de régime après accumulation de tensions
  • En écologie : les écosystèmes ne se dégradent pas graduellement, ils basculent quand un seuil de résilience est franchi
  • En psychisme : les schémas ne changent pas par ajustement continu, mais par transition quand le coût dépasse un seuil
  • En société : les transformations culturelles procèdent par accumulation puis basculement, pas par évolution lisse
Invariant

À toutes les échelles, on retrouve la même grammaire : flux + attracteurs + coûts + transitions. On ne "maîtrise" pas l'instabilité, on l'habite et on la convertit.

Falsifiabilité

Le cadre est conçu pour être falsifiable. Les tests T1-T8 définissent huit points de rupture potentiels :

  • T4 : Si ΔC n'est pas attribuable à ΔS, le canal symbolique est réfuté
  • T5 : Si une injection de S n'a pas d'effet différé, la dynamique cumulative est réfutée
  • T6 : Si une coupure de S ne fait pas chuter C (sans changement de O, R, I), la dimension symbolique est réductible
  • T7 : Si C est strictement linéaire en S (pas de point de bascule), il n'y a pas de seuil

Conclusion

Les dynamiques à seuil ne sont pas des anomalies. Elles sont la règle dans les systèmes complexes. Le cadre ORI-C fournit une méthode pour les détecter, les mesurer et les anticiper — sans prétendre les contrôler, mais en apprenant à les lire.
