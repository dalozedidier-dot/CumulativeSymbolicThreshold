# Glossary of variables and notation

This file is the contract. Resolve ambiguities here before preregistration.

- O(t): organization.
- R(t): resilience.
- I(t): integration and regulation.
- V(t): viability on [t-Δ, t], ω fixed ex ante.
- Σ(t): mismatch, Σ(t) = max(0, D(E(t)) - C(O,R,I)).
- Σ*: stress threshold.
- τ: minimum duration beyond Σ*.
- S(t): symbolic stock, α fixed ex ante.
- s(t): symbolic efficiency, ΔV/ΔS under intervention.
- C(t): order parameter, intergenerational gain due to social transmission at constant genetics over horizon T.
- T: preregistered horizon.
- ΔC(t): increment, threshold rule uses μ_ΔC, σ_ΔC, k, m.
