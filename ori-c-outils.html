Épistémologie

Tableau périodique épistémique

v1.0 — 118 éléments

Une cartographie structurée de 118 concepts épistémiques, organisés en tableau périodique par analogie avec la chimie. Chaque élément est défini selon trois niveaux d'analyse complémentaires.

Niveau 1 — Fonction · rôle opérationnel dans un raisonnement ou un système
Niveau 2 — Rôle épistémique · position dans la production de connaissance
Niveau 3 — Secteur · domaine d'application (psychisme, climat, contrôle, société)

L'outil permet de localiser rapidement la nature d'un concept, ses relations de voisinage épistémique, et les risques de confusion ou de déplacement sémantique entre secteurs.

Analyse symbolique

NEP-LIMITE 2.0

Amplification-Barometer — open source

Baromètre d'amplification symbolique fondé sur le cadre ORI-C. Mesure la dynamique de la variable d'ordre C(t) à partir de proxies observables (O, R, I) et d'un stock symbolique S(t), avec détection automatique de seuil par méthode cumulative.

Pipeline · O(t), R(t), I(t) → Cap(t) → Σ(t) → C(t)
Détection · μ + k·σ sur fenêtre glissante (k=2.5, w=20)
Verdict · ACCEPT / INDETERMINATE / REJECT avec CI 99 %
Tests · T1–T8 reproductibles, seed fixe, mode full_statistical

Inclut les tests T1 (noyau demand shock), T4 (S-riche vs S-pauvre), T7 (progressive sweep S*) et T8 (reinjection recovery). Documentation complète et artefacts reproductibles.
